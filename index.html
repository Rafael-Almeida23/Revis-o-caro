Introdução ao HTML

HTML (HyperText Markup Language) é responsável pela estrutura das páginas na web.

  • <html> - Início do documento HTML
  • <head> - Define informações do cabeçalho
  • <title> - Nome exibido na aba do navegador
  • <body> - Área visível do site
  • <h1> até <h6> - Cabeçalhos de diferentes tamanhos
  • <p> - Parágrafo de texto
  • <a> - Cria links
  • <img> - Adiciona imagens
  • <ul> / <ol> - Listas (não ordenada / ordenada)
  • <table> - Criação de tabelas
  • <div> - Container genérico
  • <span> - Container inline

Conceitos de CSS

CSS (Cascading Style Sheets) serve para personalizar a aparência do HTML.

  • color - Define a cor do texto
  • background-color - Muda a cor de fundo
  • font-size - Altera o tamanho da fonte
  • margin / padding - Espaçamento externo/interno
  • display: flex; - Layout flexível com Flexbox
  • width / height - Define largura e altura

JavaScript na Web

JavaScript permite adicionar comportamento dinâmico às páginas HTML.

  • document.getElementById("id") - Seleciona por ID
  • document.querySelector(".classe") - Seleciona usando seletor CSS
  • addEventListener("click", função) - Escuta eventos de clique
  • let valor = 10; - Declara uma variável
  • function exemplo() {} - Cria uma função
  • console.log("Texto"); - Imprime no console

Versionamento com Git & GitHub

Git é utilizado para controlar mudanças no código-fonte. GitHub é uma plataforma para hospedá-los.

  • git init - Cria um novo repositório
  • git clone URL - Copia um repositório remoto
  • git add . - Prepara arquivos para commit
  • git commit -m "mensagem" - Registra uma alteração
  • git push origin nome-da-branch - Envia as alterações
  • git pull origin nome-da-branch - Atualiza com novas alterações
